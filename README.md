# Development execution

1. Clone repository
2. Install dependencies
```
yarn install
```
3. Install Nest CLI
```
npm i -g @nestjs/cli
```

5. Clone file ```.env.template``` and rename to ```
.env```

6. Fill enviroment variables in ```.env```

7. Up app in dev:
```
yarn start:dev
```
